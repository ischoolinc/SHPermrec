# 復學通知單調整（0701）

## 修改檔案

- `休學期滿復學通知單.cs`

## 修改方法

- `GetLeaveSchoolList(int LeaveTime)`

## 變更內容

於 `QuitCode` 清單新增轉出異動代碼，使具有轉出紀錄的學生不列入復學通知單名單（`LastStatu` 設為「放棄學籍」）。

新增代碼：

| 代碼 | 說明 |
| ---- | ---- |
| 301 | 科班轉出 |
| 302 | 跳級（轉出） |
| 310 | 轉出(參與非學校型態實驗教育) |
| 311 | 轉出(舉家遷移) |
| 312 | 轉出(家長、學生調職) |
| 313 | 轉出(改變環境) |
| 314 | 轉出(其他) |
| 316 | 轉出(休學)(不計人數) |

## 測試／驗證說明

- 學生同時有休學異動與上述任一轉出代碼時，不應出現在「復學通知單(第一次休學)」或「復學通知單(第二次休學)」名單中。
- 僅有休學異動、無轉出或放棄學籍代碼者，行為應與修改前相同。
- 既有 `QuitCode`（361、367、369 等）排除邏輯未變更。
