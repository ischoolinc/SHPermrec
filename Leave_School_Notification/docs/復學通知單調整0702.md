# 復學通知單調整（0702）

## 修改檔案

- `休學期滿復學通知單.cs`

## 修改方法

- `btnExportStudent_Click`

## 匯出來源

- `lstStudent`（與列印功能相同之學生清單）

## 匯出欄位

與 `lstStudent` 欄位一致：

| 欄位 |
| ---- |
| 狀態 |
| 班級 |
| 學號 |
| 姓名 |
| 休學日期 |
| 休學原因 |
| 系統編號 |

## 使用工具方法

- `Utility.ExportXls(string ReportName, Workbook wbXls)`

## 測試／驗證說明

- 清單無學生時，應顯示「清單內沒有學生，無法匯出」並停止匯出。
- 清單有學生時，匯出 Excel 檔名應為「{報表種類}學生清單.xlsx」（例如「復學通知單(第一次休學)學生清單.xlsx」）。
- Excel 工作表名稱為「學生清單」，欄位與畫面清單一致，包含系統編號。
- 未變更列印邏輯、學生清單產生邏輯及休學／復學／放棄學籍判斷邏輯。
